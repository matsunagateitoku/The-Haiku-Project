Spring
ほの明けに花白き木や露の原
honoake ni / hana shiroki ki ya / tsuyu no hara
In the faint light of dawn / a tree blossoming in white, / the field sprinkled with dew.
寝牛とも石とも見えて草萠る
ne-ushi to mo / ishi to mo miete / kusa moyuru
A sleeping cow? / A boulder? It could be either. / Grass sprouts out.
山吹咲く工女が窓々の長屋
yamabuki saku / kōjo ga / mado-mado no / nagaya
Mountain roses bloom: / factory girls / at the windows / of a tenement house.
草をぬく根の白さ深さに堪えぬ
kusa wo nuku / ne no shirosa / fukasa ni / taene
I pull out a stalk of grass, / the root’s whiteness / and depth— / I bear with the sight.
春寒し水田の上の限なし雲
haru samushi / suiden no ue no / kagiri nashi kumo
Cold spring... / clouds without roots above / the paddy fields
赤い椿白い椿と落ちにけり
akai tsubaki / shiroi tsubaki tsubaki to / ochi ni keri
A red camellia / then a white camellia / falls down
春浅き水を渡るや鷺一つ
haru asaki / mizu wo wataru ya / sagi hitotsu
spring is shallow / crossing the water― / a single heron
Summer
行水を捨てて湖水のささ濁り
gyōzui wo / sutete kosui no / sasanigori
From a bathing tub / I throw water into the lake— / slight muddiness appears.
麦の秋盗人らしきもの通る
mugi no aki / nusubito rashiki / mono tōru
Wheat harvesting time— / a man who looks like a robber / passes by.
愕然として昼寝さめたる一人哉
gakuzen to / shite hirune same-taru / hitori kana
Startled / I wake from a midday nap / all alone.
馬ひとり忽と戻りぬ飛ぶ蛍
uma hitori / kotsu to modorinu / tobu hotaru
The horse alone / has unexpectedly returned— / flitting fireflies
空をはさむ蟹死にをるや雲の峰
kū wo hasamu / kani shini oru ya / kumo no mine
Clawing the void / lies the corpse of a crab: / mountains of clouds.
遠く高き木夏近き立てり畳む屋根に
tōku takaki / ki natsu chikaki tateri / tatamu yane ni
In the distance / a tall tree / near summertime / stands / above multitudinous roofs.
蠅打つまで蠅叩きなかりし
hae utsu / made hae-tataki / nakarishi
Until I hit the fly / the fly-swatter / did not exist.
Autumn
泣きやまぬ子に灯ともすや秋の暮
naki yamanu / ko ni hi tomosu ya / aki no kure
For a baby who / keeps crying she lights a lamp: / evening in autumn.
断食の恋ふ夜半や稲光
danjiki no / mizu kou yowa ya / inabikari
A fasting man / craves for water at midnight: / a flash of lightning.
暴動の後にまたなき月夜かな
bōdō no / ato ni mata naki / tsukiyo kana
After the riot / an incomparably beautiful / moonlit night.
相撲乗せし便船のなど時化となり
sumō noseshi / binsen no nado / shike to nari
Wrestlers are aboard / the ferry; why has it become / stormy weather?
明白む障子の蛾高し
ake shiramu / shōji no ga / takashi
Whitening with dawn, / the sliding screen: / on it a moth / high up.
父はわかつてゐた黙ってゐた庭薄
chichi ha wakatte-ita / damatte-ita / niwa susuki
Father had known, / didn't say a word: / pampas grass in the garden.
Winter
軒落ちし雪窮（港）を塞ぎけり
noki ochishi / yuki kyūkō wo / fusagi-keri
Fallen off the eaves, / a pile of snow blocks the street / in a slum area.
思はずもヒヨコ生まれぬ冬薔薇
omowazu mo / hiyoko umarene / fuyu sōbi
Unexpectedly / a chick has hatched— / midwinter rose.
牡蠣飯冷えたりいつもの細君
kaki-meshi / hie-tari / itsumo no / saikun
Oyster stew / has become cold— / same old / wife of mine.
Unclassified
ringo wo tsumami / iitsukushitemo / kurikaesaneba naranu
I pick up an apple: / I've said everything that was to be said, / But still must repeat.
omowazu mo / hiyoko umarenu / fuyu sōbi
Quite to my surprise / Some little chicks have been born: / Winter roses.
onsen no yado ni / uma no ko kaeri / hae no koe
At the hot-spring inn / A baby colt was hatched out: / The buzzing of flies.
