北村 季吟  ·  20 poems
Spring
身がろげに舞ふ羽づかひや趙飛燕
• / • / •
light as a feather / her dancing sleeves— / Zhào Fēiyàn
桃の酒いつみかはけふ三かのはら
• / • / •
the peach blossom wine— / how many cups on / this third-month plain?
匂ひみちぬ天も地もとの花ざかり
• / • / •
fragrance fills all— / heaven and earth alike / at full blossom
地主からは木の間の花の都哉
• / • / •
from Jizhu's grounds— / between the trees, the blossoms / of the capital
はら筋をよりてや笑ふ絲ざくら
• / • / •
drawing out the belly / laughing, twisted— / the weeping cherry
一僕とぼくぼくあるく花見かな
hitoshimobe to / boku boku aruku / hanami kana
just one attendant / and I plod along / blossom viewing
くず水や花の下ゆく吉野川
kuzumizu ya / hana no shita yuku / Yoshino-gawa
arrowroot water— / beneath the blossoms flows / the Yoshino River
一僕とぼくぼくありく花見哉
hitoshimobe to / boku boku ariku / hanami kana
with just one servant / I plod plod along / for blossom viewing
地主からは木の間の花の都かな
jishu kara wa / koma no hana no / miyako kana
from Jizhu's grounds / between the tree-gap blossoms— / the capital
腰をふる門の柳やかぶきもの
koshi wo furu / kado no yanagi ya / kabuki mono
swaying at the hips / at the gate, the willow— / a dandy strolling
Summer
名乗りけり大をんじゃうしほととぎす
• / • / •
it proclaimed its name— / what a mighty voice— / the cuckoo
夏痩と答へてあとは涙かな
• / • / •
summer leanness— / I answered and after that / only tears
腰をふる門やかぶきもの
• / • / •
hips swaying / at the gate—the dandy / passing by
めづらしや二四八のほととぎす
• / • / •
how rare— / the cuckoo of / the twenty-fourth day
夏痩と答へてあとは涙かな
natsuyase to / kotaete ato wa / namida kana
summer thinness— / that's my answer, and after / only tears remain
Autumn
まざまざといますがごとし魂祭
• / • / •
so vividly / as if here and present— / the spirit festival
鴈は文字おほふや霧のゐんふたぎ
• / • / •
• / • / •
秋風の姿なりけりむらすすき
aki kaze no / sugata nari keri / mura susuki
the form of the autumn wind— / that is what it is: / clusters of pampas grass
まざまざといますが如し魂祭
mazamaza to / imasu ga gotoshi / tamamatsuri
so vivid and clear / as if still present— / the souls' festival
