Hasegawa Kanajo
長谷川かな女  ·  8 poems
Spring
摩利支天の露店の上や春の月
marishiten no / roten no ue ya / haru no tsuki
above the shops / at the Marishiten Festival - / the spring moon
御所人形の稚児輪ふくらむ牡丹の芽
gosho ningyoo / no chigowa fukuramu / botan no me
the fluffy hairdo / of this Gosho doll - / bud of a peony
Autumn
重陽や椀の蒔絵のことごとし
chooyoo ya / wan no makie no / koto gotoshi
Chrysanthemum Festival - / in the bowl this laquer image / almost the same
西鶴の女みな死ぬ夜の秋
Saikaku no / onna mina shinu / yoru no aki
the heroines of Saikaku / all die in the end / autumn night
New Year
五色に塗る餅柔かしお命講
goshiki ni / nuru mochi yawarakashi / o-meikoo
the mochi cakes / in five colors and so soft ... / memorial for Saint Nichiren
大橋を恵方へ渡り詣りけり
oohashi wo / ehoo e watari / mairi keri
crossing the Great Bridge / in the auspicious direction / for the New Year
winter
貧しけれどもクリスマスの夜の父ありぬ
mazushi keredo / kurisumasu no yo no / chichi arinu
we may be poor, / but on Chirstmas Eve / Dad is home
Unclassified
Musashino no / tori kuru matsu no / shin mugen
birds of Musashino plain / coming to the pine candles - / infinity
