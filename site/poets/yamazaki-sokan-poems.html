Yamazaki Sōkan
山崎宗鑑  ·  9 poems
Spring
にがにがしいつ迄嵐ふきのたう
niganigashi / itsu made arashi fuki no tō / fuki no tō
till when will / the bitter winds blow / butterbur pot
手をついて歌申しあぐる蛙かな
te wo tsuite / uta mōshi aguru / kawazu kana
with hands supplicated / reciting a song / a frog:
満丸に出でてもながき春日かな
manmaru ni / idetemo nagaki / haru-bi kana
thou rising roundly / the sun elongates / on long spring days:
Summer
すゞしさや水に柳のかげ法師
suzushisa ya / mizu ni yanagi no / kage-bōshi
how cool- / on the water, the willows’ / silhouettes
うづきゝてねぶとに鳴けや郭公
uzuki kite / nebuto ni nake ya / Hototogisu
the deutzia moon has come / sing in a loud voice / hototogisu
月にえをさしたらばよき団哉
tsuki ni / e wo sasitaraba yoki / utiwa kana
if this moon / had a handle / a fine fan!
Winter
きえにけりこれぞまことの雪ぼとけ
kie ni keri / kore zo makoto no / yuki botoke
it has vanished— / this was the true form / of the snow Buddha
風寒し破れ障子の神無月
kaze samushi / yabure shōji no / kaminazuki
cold wind / ripped shoji screen / godless month
さむくともひになあたりそ雪仏
samuku tomo / hi ni na atari so / yuki hotoke
even if you’re cold / don’t warm yourself by the fire / snowman
