各務支考  ·  12 poems
Spring
春雨や枕崩るるうたひ本
harusame ya spring rains— / makura kuzururu the pillow of song books / utaihon collapses
spring rain— / the pillow dissolves / into the chant book
船頭の耳の遠さよ桃の花
sendo no / mimi no tosa ya / momo no hana
the boatman's / hearing so dull— / peach blossoms
馬の耳すぼめて寒し梨の花
uma no mimi / subomete samushi / nashi no hana
the horse tucks in / its ears in the cold— / pear blossoms
歌書よりも軍書にかなし吉野山
utasho yori mo / gunsho ni kanashi / Yoshino yama
more than poetry books / the war chronicles sadden: / Mount Yoshino
俳論よりも軍書にかなし　芳野山
hairon yori mo / gunsho ni kanashi / Yoshinoyama
more than a treatise on haiku — / it is a war chronicle, / Mt. Yoshino
Summer
僧達の　袂を染むる　新茶かな
So tachi no / tameto wo somuru / shincha kana
The monks / get their sleeves dyed again, / new tea.
卯の花に　さくら合はせる　明け方かな
u no hana ni / sakura awaseru / akegata kana
deutzia blossoms / meeting the cherry blossoms — / at dawn
昼顔の　砂踏み崩す　暑さかな
hirugao no / suna fumikuzusu / atsusa kana
the morning glory — / crumbling the sand underfoot, / this heat
Autumn
駒の尾に　まだ霧原の　大風引き
koma no o ni / mada kirihara no / ōkaze hiki
at the horse's tail — / still trailing the great wind / of the misty plain
参銭の　音まれまれに　落葉かな
sanzen no / oto mare mare ni / ochiba kana
the sound of offerings — / so rare, so rare, / falling leaves
Winter
食堂（じきだう）に雀啼くなり夕時雨
jikido ni / suzume naku nari / yushigure
in the refectory / a sparrow is crying— / evening winter rain
呵（しか）られて次の間へ出る寒さかな
shikarare te / tsugi no ma e deru / samusa kana
scolded out— / shuffling to the next room / in the cold
